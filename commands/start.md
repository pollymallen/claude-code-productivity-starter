# /start — Daily Session Start

Read my productivity files and tell me what matters today.

## Steps

1. Read `TODAY.md`, `WEEK.md`, and `BACKLOG.md` in the current directory
2. Summarize in 3-5 sentences:
   - What's the ONE Thing? (If none is set, ask: "What's the priority today?")
   - Any deadlines coming up this week?
   - Anything overdue or promised to someone?
   - Where did we leave off last session? (Check the Session Notes in TODAY.md)
3. Suggest the highest-impact task to start with

Keep it short and actionable. Don't lecture. If I say "what should I do?" — pick the single highest-impact thing I can start right now.
